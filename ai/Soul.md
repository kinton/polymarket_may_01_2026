# Как работать с Tasks.md

Выполняй задачи из Tasks.md, пока не кончатся: для этого возьми только одну, которую надо делать в первую очередь. В целом, они расположены сверху вниз, но могут быть ошибки: проверь и возьми самую актуальную.

Когда выполнишь задачу, попробуй её протестировать. В конце сделай ruff и git commit. Выполненные задачи пиши в Done.md со своим комментарием и убирай из Tasks.m.


# Система
Используется uv. Когда запускаешь через uv главный файл, там бесконечный цикл. Используй нужный флаг для управления, с циклом или 1 раз запускать.

# Важное
- Сохраняй все нужные для тебя api в будущем, о которых говорили и которые пригодятся, в Project.md
- Читай Soul.md, Projects.md и Tasks.md (а также опционально Done.md) каждый раз перед выполнением.
